# 四个核心功能
1. 范文学习
把公众号链接交给 AI，它会自动学习范文的写作风格，并应用于后续创作，无需反复迭代、抽卡。

2. 文章自然润色改写。内置了格式工坊的开源改写模型，提高人工创作特征，降低AI味，改写结果仅供参考，不保证能过朱雀。

3. 全流程自动处理
从写作、配图、排版到发布，一站式自动完成。
内置十套模板，预览确认后，可以直接推送到微信公众号草稿箱。

4. 学习人工修改
AI 会学习用户在微信公众号草稿箱中最后所做的人工修改与微调，并将其沉淀为规则，用于优化后续创作。
越用越顺手，逐渐形成你的专属风格。

# WeWrite 配置教程

第一次使用时照着做一遍，通常几分钟就能完成。

> **开始前准备**  
> 准备好微信公众号管理员微信、可正常登录的公众号后台，以及用于 AI 写稿和生图的平台账号。所有密钥只填写到自己电脑上的 WeWrite 设置页，不要发给他人。

## 打开 WeWrite

下载并解压文件压缩包后，双击 `Start-WeWrite.bat` 或 `WeWrite.exe`。第一次使用时打开后会出现配置界面，按下面步骤填写。

## 1. 获取公众号 AppID

1. 访问微信公众平台并登录自己的公众号。
2. 在左侧菜单进入“设置与开发” - “基本配置”。不同版本后台的菜单名称可能略有差异。
3. 找到“开发者ID（AppID）”，复制后粘贴到 WeWrite 的“公众号 AppID”。

微信公众平台：<https://mp.weixin.qq.com/>

## 2. 获取公众号 AppSecret

1. 仍在“设置与开发” - “基本配置”页面，找到“开发者密码（AppSecret）”。
2. 按页面提示查看或重置 AppSecret，通常需要公众号管理员扫码确认。
3. 复制新生成的 AppSecret，立即粘贴到 WeWrite 的“公众号 AppSecret”。

> **注意**  
> 重置 AppSecret 后，旧密钥会失效。AppSecret 相当于公众号密码，不要截图、不要通过聊天软件发送，也不要填写到不可信的网站。

## 还要配置 API IP 白名单

在 WeWrite 设置页点击“获取当前 IP”，复制显示的公网 IP。回到微信公众平台的基本配置页面，找到“IP 白名单”并加入该 IP。更换网络或公网 IP 变化后，需要重新设置，否则推送草稿可能失败。

## 3. 获取 AI 写稿 API Key（DeepSeek）

1. 打开 DeepSeek 开放平台，注册或登录账号。
2. 先在账户中充值少量余额；网页版会员和 API 账户通常是分别计费的。
3. 进入“API Keys”，点击“创建 API Key”。
4. 复制生成的密钥，粘贴到 WeWrite 的“AI 写稿 API Key”。

- DeepSeek API Keys：<https://platform.deepseek.com/api_keys>
- DeepSeek 官方 API 文档：<https://api-docs.deepseek.com/>

> **分清两件事**  
> 这里需要的是开放平台生成的 API Key，不是 DeepSeek 登录密码，也不是网页版聊天账号。API 调用会按实际用量扣费。（一般一篇写稿大概会产生 0.1 元左右的 token 费用。）

## 4. 配置豆包 Seedream 生图 API Key（选填）

> 只有需要 AI 生成正文插图或封面时才需要配置；只使用自己上传的照片可以跳过。

1. 打开火山方舟控制台，注册并完成平台要求的认证和服务开通。
2. 确认账户可使用豆包 Seedream 图片生成模型，并保持账户有可用额度。
3. 进入“系统管理” - “API Key 管理”，创建或查看 API Key。
4. 在 WeWrite 的 AI 生图配置中粘贴 API Key 即可。

- 火山方舟 API Key 管理：<https://console.volcengine.com/ark/region:ark+cn-beijing/apikey>
- Seedream 官方文档：<https://www.volcengine.com/docs/82379/seedream?lang=zh>

## 上传素材的小技巧

图片文件尽量按内容命名，例如：“领导致辞.jpg”“体验机器人环节.jpg”“颁奖合影.jpg”。

不要只使用“IMG_001.jpg”“微信图片.jpg”这类名称。清楚的文件名能帮助 AI 理解图片内容，把图片插到文章中更合适的位置。

## 配置完成后检查

- AppID、AppSecret 和 AI 写稿 API 显示“已配置”。
- 需要 AI 生图时，AI 生图 API 也显示“已配置”。
- 微信后台已经加入当前公网 IP 白名单。
- 先生成一篇短文章并推送到微信草稿箱，完成首次连通性测试。

> **密钥安全**  
> 不要把自己的 `data` 文件夹、`config.yaml`、API Key 或 AppSecret 一起发给别人。每位使用者都应填写自己的账号和密钥。
