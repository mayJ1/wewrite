WeWrite Windows 免安装版
========================

启动方法
--------
双击“启动 WeWrite.bat”或“WeWrite.exe”。
程序会启动本地服务并自动打开浏览器。

首次使用
--------
按照网页引导填写微信公众号 AppID、AppSecret、AI 写稿 API Key，
并完成账号风格问卷。所有配置只保存在本目录的 data 文件夹中。
不会配置时，请打开同目录的“WeWrite-Config-Guide.pdf”。

停止程序
--------
关闭黑色的 WeWrite 运行窗口，或在窗口中按 Ctrl+C。

数据与升级
----------
data 文件夹保存密钥、账号风格、范文、上传素材和草稿历史。
升级版本时，请保留旧版本的 data 文件夹，并复制到新版本目录。

学习草稿修改
------------
文章推送到微信草稿箱并完成修改后，回到“草稿历史”，
点击对应文章的“同步修改并学习”。系统会对比本地初稿和微信版本，
提取可复用的标题、语气、结构和措辞偏好，用于以后写稿。
学到的规则可以在草稿历史页查看和删除。

运行要求
--------
- Windows 10 或 Windows 11 64 位
- 无需安装 Python、Node.js 或 Git
- AI 写稿、AI 生图和推送微信草稿时需要联网
- 浏览和管理本地草稿时仅使用本机 127.0.0.1
